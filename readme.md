# H4X

Graphical pentesting tool using [Electron](https://www.electronjs.org/) and
[NodeJS](https://nodejs.org/es/).

![DNS Queries](./cap.png)

Supported languages:

- English
- Spanish


## Requirements

- NodeJS >= 16.14.2 LTS


## How to run

In the `app` directory run:

```bash
npm start
```


## How to build

In the `app` directory run:

```bash
npm run make
```
